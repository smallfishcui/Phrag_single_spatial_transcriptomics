# Phrag_single_spatial_transcriptomics
This folder contains the scripts used for data analysis in “Single-cell and spatial transcriptomics in Phragmites australis reveal the associations of B chromosomes with plant invasiveness,” published in Genome Biology (2026; Wang et al.).
Note that in the publication, gene names were updated to include the subgenome designation (A or D), whereas the gene names used in these scripts do not include the A or D suffix.
